# ShadowFox
## Task 1: Console-Based Calculator 
Enhanced Console-Based Calculator using Java with scientific operations, temperature conversion, and exception handling using BigDecimal for precision.
## Task 2: Simple Contact Management System (CLI)

- Console-based Java program to manage contacts
- Features:
  - Add, View, Update, Delete contacts
  - Phone number validation (numbers only)
  - Email format validation
  - Prevent duplicate phone numbers
- Built using Java, ArrayList, Scanner
